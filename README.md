# KOCKA223

Это простая веб-аппликация на HTML, CSS и JavaScript: белая пушистая кошка Луна гадает на картах Таро и отвечает на вопросы.

## Как запустить

Откройте файл [index.html](index.html) в браузере или через локальный сервер.

## Что умеет

Кошка раскладывает карты на розовом столе, показывает расклад из трёх карт и генерирует короткий ответ по вашему вопросу.
